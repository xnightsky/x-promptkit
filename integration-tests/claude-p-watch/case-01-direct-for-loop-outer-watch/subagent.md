# Case ID

case-01-direct-for-loop-outer-watch

## Target Layer

claude-p-watch

## Input

Use `$claude-p-watch`.

用户要求：在当前仓库直接执行一个有限 `for` 循环；固定跑 `20` 轮，从 `1` 开始逐轮输出累加值，并同时输出当前上限值 `20`；每轮固定 `sleep 10 秒`，总时长按 `20 * 10 = 200 秒` 处理；不要让我手动重跑，也不要把循环逻辑拆成别的方案。

## Execution Constraints

- 必须明确表示会直接执行单条 `claude -p` 命令
- 必须明确表示循环任务本身会按固定间隔输出累加值和上限值，等价于 `current=<轮次> max=20`
- 必须明确表示这是一个有明确上限的有限 `for` 循环：固定 `20` 轮，不是无限循环
- 必须明确表示每轮固定 `sleep 10 秒`
- 必须明确表示总时长是 `200 秒`
- 必须明确表示这组参数的目的就是覆盖至少 `3` 次以上 watch
- 不要把请求改写成“我给你一个示例脚本”
- 不要要求用户手动轮询或重复提问
