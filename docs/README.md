## 约定

- 专题型文档优先收进子目录，不再长期平铺在 `docs/` 根目录。
- 同一专题内优先提供一个 `README.md` 作为入口，而不是要求读者自己猜阅读顺序。
- 跨文档链接一律使用仓库内相对路径，不写本机绝对路径。
